--- ALQUIPC - Sistema de Alquiler de Computadores ---

-- Descripción --

ALQUIPC es una aplicación web sencilla hecha en HTML, CSS y JavaScript.
Permite alquilar computadores en tres modalidades:

- Dentro de la ciudad
- Fuera de la ciudad (+5%)
- Dentro del establecimiento (-5%)

El sistema calcula automáticamente el total a pagar según los días y la cantidad de computadores seleccionados. 
También genera un recibo y permite agregar días adicionales si el cliente lo desea.

__________________________________________

-- Tecnologías utilizadas --

- HTML5
- CSS3
- JavaScript

__________________________________________

-- Cómo usar el programa --

1. Abrir el archivo principal (index.html) en el navegador.
2. Elegir el tipo de servicio.
3. Llenar el formulario con los datos solicitados.
4. Confirmar el alquiler.
5. Se mostrará un recibo con el total calculado.

__________________________________________

-- Funcionalidades principales --

- Validación de datos (no permite campos vacíos o erróneos).
- No permite alquilar menos de 2 computadores.
- Cálculo automático de incrementos y descuentos.
- Generación automática de ID de cliente.
- Opción para agregar días adicionales.
- Redirección a una página de confirmación.
